Feature Flag Overview

This interface allows you to manage feature flags at different scopes:

  • Roles: Default settings for user roles (Anonymous, Muso, Jam Manager)
  • Users: Per-user overrides for specific individuals
  • Jams: Per-jam settings that apply to specific jam sessions
